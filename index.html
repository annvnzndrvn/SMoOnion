SMoOnion

Stop-Motion software for DSLR cameras with onion skin features.

Currently only Nikon D90 supported.

Features roadmap:

Single frame snapshot LiveView with previous frame on onion skin Battery level warning Onion skin visible up to 3 previous frames Import video for rotoscoping Support for rest of cameras

Known issues and improvements:

Onion skin on last frame not getting correct opacity level Improve timer logic Every time user takes a snapshot, next time it takes slightly longer to process
